Randomizer
==========

![screenshot](screenshot.png)

Random symbol sequence (e.g. password) generator for Haiku

Build Instructions:
  * Clone the repo and run the following:
  ```
  cd Randomizer/src
  make
  ./Randomizer
  ```

Requirements:
* 1. BeOS R5 or newer
