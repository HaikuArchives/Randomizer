Randomizer
==========

![screenshot](screenshot.png)

Random symbol sequence (e.g. password) generator for Haiku

Build Instructions:
  * Clone the repo and `cd` to the folder where you cloned it and run the following:
  ```
  cd src
  make
  ./Randomizer
  ```

Requirements:
* 1. BeOS R5 or newer
